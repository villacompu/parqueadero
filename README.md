# Parking PH - Primitiva Parque Natural (Streamlit)

Proyecto modular (carpetas `auth/`, `db/`, `services/`, `views/`) para facilitar mantenimiento.

## Ejecutar
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Base de datos
- Por defecto usa `parking.db` en la carpeta del proyecto.
- Evidencias/fotos se guardan en `uploads/`.
